# Category: Foreign Affairs

استقبل وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة دمشق رئيس وفد وحدة مكافحة الإرهاب التابعة للاتحاد الأوروبي السيد بارتجان وجتر.
حيث جرى خلال اللقاء بحث سبل تعزيز التعاون بين الجمهورية العربية السورية والاتحاد الأوروبي في مجال مكافحة الإرهاب والتحديات الأمنية المشتركة، كما تم التأكيد على أهمية مواصلة الحوار البنّاء وتطوير قنوات التواصل بما يخدم أمن واستقرار المنطقة.
