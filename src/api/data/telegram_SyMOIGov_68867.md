رئيس الجمهورية العربية السورية السيد أحمد الشرع يتقبّل أوراق اعتماد سعادة السفير عبد القادر قاسم الحسني، سفير الجمهورية الجزائرية الديمقراطية الشعبية لدى الجمهورية العربية السورية، وذلك بحضور وزير الخارجية والمغتربين السيد أسعد الشيباني.
