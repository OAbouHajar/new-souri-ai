السيد الرئيس أحمد الشرع يلتقي المدير العام لمنظمة الصحة العالمية الدكتور تيدروس أدهانوم، وذلك بحضور وزير الخارجية والمغتربين السيد أسعد حسن الشيباني، على هامش أعمال مؤتمر قمة المناخ ( cop30 ) المنعقد في مدينة بيليم البرازيلية.
