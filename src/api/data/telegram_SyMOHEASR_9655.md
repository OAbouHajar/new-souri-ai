وزير التعليم العالي والبحث العلمي الدكتور "مروان الحلبي" يترأس اجتماع مجلس إدارة المستشفى الوطني الجامعي بدمشق، مؤكداً أهمية استثمار العنصر البشري، ورسم خارطة تطوير لعمل المستشفى للعام القادم، وتحسين جودة الخدمات المقدمة، وتعزيز الرقمنة، والشراكات البحثية والتعليمية لسد الثغرات والاختصاصات المجتمعية.
