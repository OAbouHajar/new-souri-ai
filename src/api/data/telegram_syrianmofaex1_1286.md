لقاء ثنائي جمع وزير الخارجية والمغتربين السيد أسعد حسن الشيباني ونظيره التركي السيد هاكان فيدان، وذلك على هامش الاجتماع الوزاري الطارئ لمنظمة التعاون الإسلامي لمناقشة العدوان الاسرائيلي المتواصل على الشعب الفلسطيني، المنعقد في مدينة جدة.
