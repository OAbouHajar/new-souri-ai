بحضور رئيس هيئة الأركان العامة اللواء علي النعسان، أقامت الفرقة 56 حفل تخريج دورة رفع مستوى، وذلك ضمن برنامج التدريب العسكري والبدني والتكتيكي الهادف إلى رفع جاهزية المقاتلين وتعزيز كفاءتهم القتالية، وفق الخطة المعتمدة لتأهيل القوات المسلحة.
