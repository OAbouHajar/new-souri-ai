وزير التعليم العالي الدكتور "مروان الحلبي" يترأس الاجتماع الأول لمجلس مفوضي الهيئة الوطنية للجودة والاعتمادية لعام 2025.

ناقش الاجتماع استراتيجية جديدة لتعزيز التكامل بين التعليم العالي والتعليم قبل الجامعي، وبناء منظومة وطنية لضمان استدامة جودة التعليم وربط مخرجاته بالتنمية الوطنية، مع تحديد خطة العمل السنوية للعام 2026.
