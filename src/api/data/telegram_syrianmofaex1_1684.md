# Date: 2025-10-22

استقبل وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة دمشق سفير كندا لدى الجمهورية العربية السورية السيد غريغوري غاليغان.
وخلال اللقاء تسلّم السيد الوزير أوراق اعتماد السفير غاليغان كسفير دائم غير مقيم، وجرى بحث العلاقات الثنائية بين البلدين وسبل تطويرها في مختلف المجالات، إلى جانب تبادل وجهات النظر حول عدد من القضايا ذات الاهتمام المشترك.
