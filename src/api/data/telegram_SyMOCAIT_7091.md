عقد وزير الاتصالات وتقانة المعلومات السيد "حسين المصري" والكادر الإداري معه، اجتماعًا عبر الإنترنت مع وزارة الاتصالات وتقنية المعلومات في المملكة العربية السعودية، تناول الاجتماع سبل التعاون المشترك في قطاع الاتصالات بهدف تحسين جودة الخدمات وتبادل المعرفة والتجارب التقنية.
