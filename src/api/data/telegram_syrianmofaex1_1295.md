وزير الخارجية والمغتربين السيد أسعد حسن الشيباني يلتقي في العاصمة دمشق عضو البرلمان البريطاني ووزير الدولة لشؤون الشرق الأوسط وشمال إفريقيا السيد هاميش فالكونر.

جرى خلال اللقاء بحث سبل تعزيز العلاقات الثنائية بين الجمهورية العربية السورية والمملكة المتحدة، إضافة إلى مناقشة القضايا الإقليمية والدولية ذات الاهتمام المشترك.
