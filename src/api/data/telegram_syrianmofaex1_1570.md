# Category: Foreign Affairs
# Date: 2025-09-26

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني بوزير الدولة للشؤون الخارجية وعضو مجلس الوزراء والمبعوث الخاص لشؤون المناخ في المملكة العربية السعودية السيد عادل الجبير، وذلك على هامش أعمال الدورة الحالية للجمعية العامة للأمم المتحدة في نيويورك.
