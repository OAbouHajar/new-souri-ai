وزير الخارجية والمغتربين السيد أسعد حسن الشيباني يرفع علم الجمهورية العربية السورية فوق مبنى السفارة السورية في العاصمة البريطانية لندن، إيذانًا باستئناف عملها الرسمي بعد إغلاقها 14 عامًا.
