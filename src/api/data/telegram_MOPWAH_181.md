بالتعاون مع اليونيسيف، تقوم الشركة العامة للدراسات الهندسية بإعادة تأهيل أربع مدارس في حمص، تشمل أعمالاً إنشائية ومعمارية وكهربائية وصحية، وفق الكود السوري واشتراطات المنظمة.

 بدأ العمل في فبراير 2025 ولا يزال مستمرًا تحت إشراف الشركة، مع تجهيز المدارس بدورات مياه لذوي الاحتياجات الخاصة، طاقة شمسية، وتأهيل الملاعب والبنى التحتية.
