# Category: Interior
# Date: 2025-11-23

عزّزت وحدات الأمن الداخلي انتشارها في مختلف أحياء ومداخل مدينة حمص، وفرضت طوقاً أمنياً محكماً ترافق مع تسيير دوريات راجلة وآلية، تزامناً مع تطبيق حظر تجوال مؤقت. 

وتأتي هذه الإجراءات ضمن خطة أمنية تهدف إلى منع أي تجاوزات، وترسيخ الأمن والاستقرار في المحافظة، وصون الممتلكات العامة والخاصة.
