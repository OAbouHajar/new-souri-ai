# Source: Ministry of Foreign Affairs and Expatriates
# Category: Foreign Affairs
# Date: 2025-11-01
# Link: https://t.me/syrianmofaex1/1729

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة البحرينية المنامة مستشارة رئيس الجمهورية الفرنسية السيدة آن كلير ليجيندر والوفد المرافق لها، وذلك على هامش مؤتمر حوار المنامة 2025.
وجرى خلال اللقاء بحث العلاقات الثنائية بين البلدين وسبل تعزيزها، والقضايا ذات الاهتمام المشترك.
