استقبل وزير الخارجية والمغتربين السيد أسعد حسن الشيباني اليوم وزيرة التجارة الخارجية والتنمية الهولندية أوكجيه دي فريز والوفد المرافق لها.
وجرى خلال اللقاء بحث سبل تعزيز العلاقات الثنائية والتعاون الاقتصادي والتنموي بين الجمهورية العربية السورية ومملكة هولندا، إلى جانب تبادل وجهات النظر حول القضايا ذات الاهتمام المشترك.
