🚨 تحذير عاجل | صفحات وروابط مزيفة تنتحل اسم شام كاش 🚨

نحذّر جميع المستخدمين الكرام من صفحات وروابط مزيفة تنتحل اسم شام كاش وتروّج لبطاقات “فيزا شام كاش” مزعومة بهدف سرقة بيانات المستخدمين وأموالهم.

⚠️ يرجى عدم التفاعل مع هذه الصفحات أو إدخال أي معلومات مالية أو شخصية عبرها.

🔒 تذكير هام:

شام كاش لا تطلب منكم إدخال بياناتكم أو إصدار بطاقات عبر روابط خارجية.

* لا تقوم شام كاش بإرسال رسائل خاصة أو منشورات تتضمن روابط للتسجيل أو إصدار بطاقات.

* تعاملوا فقط مع القنوات الرسمية أعلاه لحماية بياناتكم وأموالكم.
