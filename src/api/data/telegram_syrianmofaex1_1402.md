# Category: Foreign Affairs
# Date: 2025-09-14

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني بنائب رئيس الوزراء ووزير الخارجية الأردني السيد أيمن الصفدي، والمبعوث الخاص للولايات المتحدة الأمريكية إلى سوريا السيد توماس باراك، وذلك على هامش أعمال الاجتماع الوزاري للقمة العربية الإسلامية الطارئة المنعقدة في دولة قطر.
