# Source: Ministry of Foreign Affairs and Expatriates
# Category: Foreign Affairs
# Date: 2025-10-10
# Link: https://t.me/syrianmofaex1/1622

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني ووزير العدل السيد مظهر الويس ورئيس جهاز الاستخبارات العامة السيد حسين السلامة في بيروت بفخامة رئيس الجمهورية اللبنانية السيد جوزاف عون، وذلك ضمن الزيارة الرسمية التي يقوم بها الوفد إلى لبنان.
وتناول اللقاء آخر المستجدات الإقليمية، وسبل تعزيز التنسيق بين دمشق وبيروت، بما يخدم مصالح البلدين والشعبين الشقيقين.
