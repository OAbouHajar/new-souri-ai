فخامة رئيس الجمهورية العربية السورية السيد أحمد الشرع، يرافقه وزير الخارجية والمغتربين السيد أسعد حسن الشيباني يلتقي برئيسة وزراء إيطاليا السيدة جورجيا ميلوني، وذلك على هامش أعمال الدورة الثمانين للجمعية العامة للأمم المتحدة بنيويورك.
