# Date: 2025-11-09

اجتمع السيد الرئيس أحمد الشرع اليوم في مقرّ صندوق النقد الدولي بواشنطن مع السيدة كريستالينا جورجييفا، المديرة العامة للصندوق، وذلك بحضور وزير الخارجية والمغتربين السيد أسعد الشيباني.

وتم خلال اللقاء بحث مجالات التعاون الممكنة بين سوريا وصندوق النقد الدولي، بما يسهم في تعزيز مسار التنمية ودعم الجهود لتطوير الواقع الاقتصادي.
