# Category: Foreign Affairs

استقبل وزير الخارجية والمغتربين السيد أسعد حسن الشيباني اليوم في دمشق المبعوث الخاص للأمم المتحدة إلى سوريا السيد غير بيدرسون والوفد المرافق له.

تناول الجانبان التطورات الأخيرة في المنطقة، كما تم التأكيد على احترام سيادة سوريا ووحدة أراضيها، وضرورة تهيئة الظروف المناسبة لعودة اللاجئين وتحسين الوضع الإنساني.
