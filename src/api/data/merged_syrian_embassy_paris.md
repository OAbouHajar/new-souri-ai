**الخاصة الحالت ولبعض الوكالة نوع حسب إضافية وثائق طلب يمكن -٤**

**)السورية الخارجية.**

**3: عن صادرة وثيقة ابراز أو )الميكانيك( األصلي السيارة دفتر إحضار فيجب سيارة بيع بخصوص الوكالة كانت إذا**
