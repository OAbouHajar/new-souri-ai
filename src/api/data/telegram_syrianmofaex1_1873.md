وزير الخارجية والمغتربين السيد أسعد حسن الشيباني يشارك في فعالية حوارية نظمها المعهد الملكي للشؤون الدولية "تشاتم هاوس" في العاصمة البريطانية لندن، تحت عنوان «رؤية ودور السياسة الخارجية لدى سوريا الجديدة». حيث أدار الحوار السيد سايمون فريزر رئيس المعهد.
