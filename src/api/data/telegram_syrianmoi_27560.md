# Source: Syrian Ministry of Interior
# Category: Interior
# Date: 2025-12-08
# Link: https://t.me/syrianmoi/27560

تواصل وزارة الداخلية تعزيز إجراءاتها الأمنية تزامنًا مع الاحتفالات الشعبية التي تشهدها المحافظات إحياءً لذكرى التحرير.

تركّز الوزارة على تأمين سير الفعاليات بشكل متكامل، من خلال مراقبة نقاط التجمع الحيوية، وتنظيم حركة المشاركين، وضمان جاهزية الفرق الأمنية للتدخل الفوري عند الحاجة، بما يوفّر بيئة آمنة ومهيّأة لإحياء المناسبة الوطنية بكل انضباط.
