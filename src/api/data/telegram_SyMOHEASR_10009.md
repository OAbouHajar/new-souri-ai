لقاء وزير التعليم العالي والبحث العلمي الدكتور "مروان الحلبي" مع وفد من الحكومة البريطانية برئاسة السيدة "آن سنو" الممثلة الخاصة للمملكة المتحدة لشؤون سوريا، والسيدة "ماري شو كليدج" نائبة رئيس البعثة، لمناقشة تعزيز التعاون الأكاديمي بين سوريا والمملكة المتحدة، بالاضافة إلى تفعيل برنامج تشفينينغ للمنح الدراسية، ودعم مراكز اللغة الإنجليزية وتطوير التحول الرقمي والتعاون مع القطاع الخاص لإنشاء مراكز تعليمية جديدة.
