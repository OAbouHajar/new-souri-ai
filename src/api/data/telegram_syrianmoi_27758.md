# Category: Interior
# Date: 2025-12-20
# Link: https://t.me/syrianmoi/27758

أحبطت إدارة مكافحة المخدرات، إثر متابعة ميدانية دقيقة ورصد استخباراتي مستمر، محاولة تهريب كمية كبيرة من حبوب الكبتاغون، كانت مخبأة داخل كراج البولمان في مدينة حماة ومعدة للإرسال خارج البلاد باتجاه المملكة العربية السعودية.

وقدّرت الكمية المضبوطة بنحو 26 ألف حبة، مخبأة داخل سرير معدني ومموّهة ضمن مفرش خشبي بطريقة احترافية، ما يبرز مستوى الجاهزية والكفاءة الأمنية لدى الجهات المعنية.

واتخذت الجهات المعنية الإجراءات القانونية اللازمة أصولًا، وصودرت المضبوطات، مع استمرار التحقيق لكشف جميع المتورطين وتقديمهم إلى القضاء المختص.
