# Source: Syrian Ministry of Interior
# Category: Interior
# Date: 2025-11-23
# Link: https://t.me/syrianmoi/27216

تعزيز أمني تشهده بلدة زيدل وعدة مناطق جنوب مدينة حمص، حيث باشرت قوى الأمن الداخلي رفع مستوى الجاهزية وتنفيذ انتشار مكثف داخل هذه المناطق ومحيطها، لضمان الأمن وحماية الاستقرار ومنع أي استغلال للحادثة لإثارة الفتنة، عقب وقوع جريمة قتل رجل وزوجته.

وتواصل الجهات المختصة تنفيذ الإجراءات القانونية وجمع الأدلة لتحديد الجناة وملاحقتهم، داعيةً المواطنين إلى التعاون والالتزام بالتوجيهات الرسمية.
