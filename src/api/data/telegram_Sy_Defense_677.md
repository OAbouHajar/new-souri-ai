# Category: Defense
# Date: 2025-07-01

نفّذ الخريجون عرضًا عسكريًا مهيبًا تضمّن مسير النظام المُنضم، واستعراض حركات السلاح بتنسيق دقيق، إضافةً إلى عرضٍ للقوة والقتال القريب، عَكَسَ مستوى الجاهزية والانضباط الذي وصلت إليه الكوادر المُتخرّجة، ضمن رؤية تهدف إلى إعداد جيلٍ عقائدي ملتزم بمهامه.
