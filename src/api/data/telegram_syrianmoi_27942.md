# Category: Interior
# Date: 2025-12-25

أسلحة متنوعة وذخائر مختلفة، ضبطتها الوحدات المختصّة في محافظة حلب، بالتعاون مع جهاز الاستخبارات العامة خلال العملية الأمنية، حيث جرى التحفظ على المضبوطات ونقلها إلى الجهات المختصة لاتخاذ الإجراءات القانونية اللازمة بحق المتورطين.

وتأتي هذه العملية في إطار الجهود المستمرة لتفكيك الخلايا الإرهابية ومنع أي تهديد يمس أمن المواطنين وسلامة المجتمع، بما يعكس قدرة الأجهزة على مواجهة التهديدات الإرهابية وحماية الأمن المجتمعي.
