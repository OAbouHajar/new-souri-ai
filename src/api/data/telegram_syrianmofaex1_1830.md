# Source: Ministry of Foreign Affairs and Expatriates
# Category: Foreign Affairs
# Date: 2025-11-10
# Link: https://t.me/syrianmofaex1/1830

التقى السيد الرئيس أحمد الشرع اليوم في البيت الأبيض نظيره الأمريكي السيد دونالد ترامب، وذلك بحضور وزير الخارجية والمغتربين السيد أسعد حسن الشيباني ونظيره الأمريكي السيد ماركو روبيو، والمبعوث الأمريكي الخاص إلى سوريا السيد توماس باراك.
وناقش الجانبان خلال اللقاء العلاقات الثنائية بين الجمهورية العربية السورية والولايات المتحدة الأمريكية وسبل تعزيزها إضافةً إلى عددٍ من القضايا الإقليمية والدولية ذات الاهتمام المشترك.
