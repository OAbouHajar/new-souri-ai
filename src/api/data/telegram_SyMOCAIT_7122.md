التقى وزير الاتصالات وتقانة المعلومات المهندس "حسين المصري" مع القائم بأعمال سفارة جمهورية قبرص لدى الجمهورية العربية السورية السيد "ميخاليس خادجيكيرو" وبحث معه سبل تعزيز التعاون بين البلدين في مجال الاتصالات وتكنولوجيا المعلومات وفرص تطوير العلاقات الثنائية في هذا القطاع الحيوي، بما يسهم في تطوير البنية التحتية الرقمية في سورية.
