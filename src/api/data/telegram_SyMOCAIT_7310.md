وقّع معاون وزير الاتصالات وتقانة المعلومات السيد "عماد الدين حمد" اتفاقيتين مع رئيسي فرعي نقابتي الأطباء وأطباء الأسنان في محافظة طرطوس، وتهدف الاتفاقيتان إلى توطين رواتب العاملين الإداريين والأطباء ضمن المؤسسة السورية للبريد مما يسهّل استلام المستحقات المالية عبر المكاتب البريدية أو توصيلها إلى العناوين المحددة بواسطة مراسلي البريد، مما يعزز كفاءة الخدمات البريدية ويخدم مختلف شرائح المجتمع.
