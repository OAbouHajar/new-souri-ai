أصدرت وزارة التربية اليوم قائمة جديدة تتضمن الموافقة على تحديد مركز ‏عمل 26 عاملاً ممن تقدموا بطلباتهم عبر المنصة الإلكترونية لخدمات ‏العاملين وانطبقت عليهم المعايير والشروط المحددة. ‏

وأكدت الوزارة ضرورة مراجعة ‏المتقدمين بطلباتهم عبر المنصة لحساباتهم الشخصية بشكل دوري لمعرفة ‏حالة طلباتهم واستدراك أي نقص بالثبوتيات المطلوبة.
