أجرى وزير الاتصالات وتقانة المعلومات السيد "عبد السلام هيكل" زيارة إلى المؤسسة السورية للبريد، لبحث آليات تحسين جودة الخدمات البريدية ورفع كفاءتها بما يلبي متطلبات المواطنين، إلى جانب مناقشة خطط التطوير الشامل وتوسيع نطاق الخدمات المقدمة، وتعزيز التنسيق مع الوزارات والجهات العامة لضمان تكاملها وتقديمها عبر منصة مركزية موحدة.
