# Category: Foreign Affairs

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني، يرافقه رئيس جهاز الاستخبارات العامة السيد حسين السلامة اليوم في العاصمة الصينية بكين عضو المكتب السياسي للجنة المركزية للحزب الشيوعي الصيني وأمين اللجنة المركزية للشؤون السياسية والقانونية السيد تشين وين تشينغ.
وجرى خلال اللقاء بحث سبل تعزيز التعاون الأمني وتطوير آليات التنسيق المشترك بين البلدين بما يخدم المصالح الاستراتيجية المشتركة.
