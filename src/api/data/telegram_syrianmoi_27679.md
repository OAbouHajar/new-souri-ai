# Source: Syrian Ministry of Interior
# Category: Interior
# Date: 2025-12-16
# Link: https://t.me/syrianmoi/27679

العملية الأمنية التي نفّذتها وحدات وزارة الداخلية، بالتعاون مع جهاز الاستخبارات العامة في محافظة إدلب، استهدفت موقع خلية إرهابية تتبع لتنظيم "داعش" الإرهابي، بعد متابعة دقيقة ورصد مستمر لتحركات عناصرها.

أسفرت العملية عن إلقاء القبض على جميع أفراد الخلية وعددهم ثمانية، و حُيد العنصر التاسع، متزعم الخلية، أثناء المداهمة. كما عُثر على أسلحة وذخائر متنوعة.
