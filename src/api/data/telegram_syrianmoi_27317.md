# Source: Syrian Ministry of Interior
# Category: Interior
# Date: 2025-11-28
# Link: https://t.me/syrianmoi/27317

أكدت وزارة الداخلية من خلال انتشار وحداتها في كافة المحافظات التزامها الكامل بحماية المشاركين وضمان تنظيم المسيرات الجماهيرية بأعلى معايير السلامة والانضباط.

تهدف هذه الإجراءات لتعزيز بيئة آمنة للتعبير الوطني وترسيخ الثقة بين المواطنين والدولة مع الحفاظ على النظام العام واستقرار المجتمع.
