# Category: Foreign Affairs

التقى فخامة رئيس الجمهورية السيد أحمد الشرع بفخامة الرئيس رجب طيب أردوغان رئيس الجمهورية التركية، بحضور وزير الخارجية والمغتربين السيد أسعد حسن الشيباني، وذلك على هامش أعمال القمة العربية الإسلامية الطارئة في الدوحة.
