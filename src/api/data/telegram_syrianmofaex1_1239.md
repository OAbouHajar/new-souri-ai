وزير الخارجية والمغتربين السيد أسعد حسن الشيباني برفقة وزير الدفاع اللواء المهندس مرهف أبو قصرة ورئيس جهاز الاستخبارات السيد حسين سلامة يلتقون بوزير الدفاع التركي السيد يشار غولر، حيث تم توقيع اتفاقية تدريب واستشارات عسكرية بين سوريا وتركيا، تشمل دورات تدريبية وبرامج ومساعدات فنية تهدف لتعزيز وتطوير دفاعات الجيش العربي السوري.
