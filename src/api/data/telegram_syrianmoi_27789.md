# Category: Interior
# Date: 2025-12-21

أسلحة متنوعة وذخائر مختلفة، وأموال مُعدّة لتمويل أنشطة إرهابية، ضبطتها قيادة الأمن الداخلي في محافظة ريف دمشق، بالتعاون مع جهاز الاستخبارات العامة، خلال عملية أمنية استهدفت تفكيك خلية إرهابية تتبع تنظيم داعش الإرهابي في مدينة داريا بريف المحافظة.

صودرت المضبوطات ونُقلت إلى الجهات المختصة لاتخاذ الإجراءات القانونية اللازمة.
