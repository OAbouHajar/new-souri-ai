# Category: Foreign Affairs
# Date: 2025-08-05
# Link: https://t.me/syrianmofaex1/1203

استقبل وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة دمشق السيدة ستيفاني ماكولم سفيرة كندا لدى الجمهورية اللبنانية.

وجرى خلال اللقاء استعراض العلاقات الثنائية، وسبل تعزيز التواصل بين البلدين في الملفات ذات الاهتمام المشترك، وأكد معالي الوزير على أهمية احترام سيادة سوريا ووحدتها وضرورة إقامة حوار بنّاء قائم على المصالح المشتركة والاحترام المتبادل.
