# Category: Foreign Affairs

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة العمانية مسقط وزير التجارة والصناعة وترويج الاستثمار السيد قيس اليوسف.
وتناول الجانبان خلال اللقاء سبل تعزيز التبادل التجاري بين البلدين الشقيقين، والتعاون في عملية إعادة بناء سوريا الجديدة.
