# Category: Foreign Affairs
# Date: 2025-08-04

ترأست وزارة الخارجية السورية سلسلة المناقشات التي جرت مع وفد الموظفين الأمريكيين برئاسة السيد نيكولاس جرينجر، مدير برنامج المنصة الإقليمية السورية (SRP)، وذلك بمشاركة وزارات الداخلية والمالية والاتصالات والبنك المركزي السوري، إلى جانب عدد من الجهات المعنية. وتركزت المناقشات حول عدد من القضايا السياسية والاقتصادية والإنسانية والدبلوماسية ذات الاهتمام المشترك.
