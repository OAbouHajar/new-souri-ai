وسط الظروف الجوية الصعبة وتساقط الثلوج، تتابع وزارة الداخلية عملها بشكل مستمر، وتتخذ كافة الإجراءات اللازمة لضمان سلامة المواطنين.

كما تدعو الجميع إلى الالتزام بالتعليمات الصادرة، والقيادة بحذر، والتواصل مع الجهات المختصة عند الضرورة.
