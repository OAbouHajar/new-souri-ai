# Category: Foreign Affairs
# Date: 2025-10-11

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني بإدارة التعاون الدولي في وزارة الخارجية ومديري التعاون الدولي في المحافظات والوزارات والجهات العامة.
وجرى خلال الاجتماع بحث آليات تفعيل وتطوير التعاون الدولي بين الجمهورية العربية السورية وسائر الدول والمنظمات الدولية، إضافة إلى مناقشة الخطط والبرامج المشتركة الهادفة إلى تعزيز الشراكات التنموية والاقتصادية ودعم القطاعات الخدمية.
