البريد السوري يوسع تعاونه الإقليمي عبر مذكرتي تفاهم جديدتين

دائرة الإعلام- 30/9/2025
وقعت المؤسسة السورية للبريد ممثلةً بمديرها العام السيد "عماد الدين حمد" مذكرتي تفاهم مع كل من مؤسسة البريد التركي ومجموعة بريد المغرب، حيث مثّل مؤسسة البريد التركي مديرها العام السيد "هاكان غولتان"، ووقع السيد "أحمد أمين التويمي" المدير العام لمجموعة بريد المغرب ممثلاً عن المجموعة.
