التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة الأمريكية واشنطن اللجنة الأمريكية للحريات الدينية العالمية، وبحث معها سبل تعزيز الحوار والتفاهم بين الشعوب والثقافات.
