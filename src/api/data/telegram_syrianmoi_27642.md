# Category: Interior
# Date: 2025-12-11

افتتح معاونُ وزير الداخلية للشؤون المدنية، العميد زياد فواز العايش، مركزَ الأحوال المدنية الجديد في حي الحمدانية بمدينة حلب، وذلك برفقة محافظ المدينة، المهندس عزام الغريب.

ويتميّز المركز بتجهيزات تقنية متقدمة تشمل منظومات تسجيل حديثة، ونوافذ خدمة متعددة، وقاعات انتظار مُهيّأة لاستقبال المراجعين، إضافة إلى بنية معلوماتية مؤتمتة تسهم في تسريع إنجاز المعاملات وضمان دقة البيانات. كما تضمن تصميمُ المركز توفيرَ بيئةِ عملٍ تنظيميةٍ تستجيب لمتطلبات الخدمة العامة، وتسهّل حركة المراجعين ضمن مسارٍ واضح وفعّال.

ويأتي افتتاح هذا المركز في إطار الخطة التطويرية التي تنفذها وزارة الداخلية للارتقاء بمنظومة الخدمات المدنية، وتحديث البنية التقنية والإدارية في دوائرها، بما يضمن تبسيط الإجراءات وتعزيز جودة الخدمات المقدّمة للمواطنين.
