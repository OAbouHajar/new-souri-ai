وزير الخارجية والمغتربين السيد أسعد حسن الشيباني يجتمع بنائب رئيس الوزراء الروسي السيد ألكساندر نوفاك، وذلك في إطار زيارة الوفد الروسي لإجراء محادثات موسعة حول سبل تعزيز التعاون الثنائي بين البلدين.
