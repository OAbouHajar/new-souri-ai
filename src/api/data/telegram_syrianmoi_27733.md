# Date: 2025-12-18

المركز الجديد للأحوال المدنية في حي الحمدانية بحلب يوفّر بيئة خدمية متكاملة بمعايير حديثة، مزوّد بأنظمة تقنية متطورة ونوافذ خدمة متعددة، مع مراعاة ذوي الاحتياجات الخاصة، بما يضمن سرعة وكفاءة في إنجاز المعاملات.
