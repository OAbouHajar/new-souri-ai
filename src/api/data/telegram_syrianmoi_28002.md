وزير الداخلية المهندس أنس خطاب عبر منصة (X):

منذ الأيام الأولى للتحرير أثبتت وزارة الداخلية أنها الدرع الحصين لأهلنا في مواجهة التحديات الأمنية، بمختلف أشكالها وصورها، وذلك عبر التعاون والتنسيق مع باقي الوزارات والمؤسسات وعموم شعبنا العظيم.

ومنذ البداية سعينا لتقديم صورة جديدة للأمن ومفهومه وأنه لحماية الناس وأمانهم لا لتخويفهم وترويعهم، وقمنا كذلك بإقرار مدونة السلوك التي تضبط العمل وتؤطره وفق الأسس القانونية والضوابط اللازمة.
