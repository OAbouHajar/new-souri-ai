# Source: Ministry of Foreign Affairs and Expatriates
# Category: Foreign Affairs
# Date: 2025-08-20
# Link: https://t.me/syrianmofaex1/1272

رفع علم الجمهورية العربية السورية فوق مبنى السفارة السورية في العاصمة الليبية طرابلس، وذلك إيذانًا ببدء الإجراءات العملية لإعادة افتتاح السفارة واستئناف عملها بشكل رسمي.
