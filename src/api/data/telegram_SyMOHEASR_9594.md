**اجتماع لمناقشة تطوير معايير الاعتماد الأكاديمي للجامعات الخاصة في سوريا**.

عُقد اجتماع برئاسة الدكتور "محمد السويد"، معاون وزير التعليم العالي والبحث العلمي لشؤون التعليم الخاص، وبحضور مدير مؤسسات التعليم الخاصة الدكتور "غذوان علي" مع عددٍ من مالكي الجامعات الخاصة السورية، لمناقشة سبل تطوير التعليم الجامعي الخاص وتعزيز معايير الاعتماد الأكاديمي.
