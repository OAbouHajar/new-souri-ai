الهيئة الوطنية لخدمات تقانة المعلومات 
مركز أمن المعلومات 
قسم التوعية السيبرانية 

في عالم التهديدات الإلكترونية المتطورة، لم يعد الاعتماد على حل أمني واحد كافياً لحماية بياناتك وأنظمتك.
الاستثمار في الحماية أصبح ضرورة وليست ترفاً.

🔒 مفهوم الدفاع المعمق (7 طبقات حماية):

👥 1. طبقة الإنسان (Human Layer)

· أهم عناصر الحماية وأكثرها تأثيراً
· التدريب المستمر على الممارسات الآمنة
· التوعية بمخاطر التصيد الإلكتروني

🌐 2. طبقة المحيط (Perimeter Layer)

· جدران الحماية (Firewalls)
· أنظمة كشف الاختراق
· أنظمة منع الاختراق

🔗 3. طبقة الشبكة (Network Layer)

· تقسيم الشبكات (Network Segmentation)
· مراقبة حركة البيانات
· كشف الأنشطة غير الطبيعية

💾 4. طبقة البيانات (Data Layer)

· تشفير البيانات الحساسة
· نسخ احتياطي منتظم
· إدارة صلاحيات الوصول

⚙️ 5. طبقة التطبيقات (Application Layer)

· اختبار اختراق التطبيقات
· مراجعة الأكواد البرمجية
· تحديث التطبيقات باستمرار

🎯 6. طبقة الأصول الحرجة (Mission Critical Assets)

· حماية الخوادم الرئيسية
· أنظمة المراقبة المستمرة
· خطط استعادة الكوارث

💻 7. طبقة نقاط النهاية (End Point Layer)

· حماية الأجهزة الشخصية
· برامج مكافحة الفيروسات
· تحديث الأنظمة باستمرار

📝 خلاصة:
الحماية الفعالة تتطلب:

· ✅ تكامل الحلول الأمنية
· ✅ المراقبة المستمرة
· ✅ التحديث الدوري
· ✅ التوعية المستدامة
