لمراقبة جودة الخبز ووزنه، وعدد الأرغفة، قام مدير التجارة الداخلية وحماية المستهلك في دير الزور، الأستاذ "خليل الصالح"، بجولة تفقدية فجر اليوم شملت الأفران وصولاً إلى الريف الشرقي.

وأكد "الصالح" أن الجولات الرقابية ستستمر لضمان وصول المادة إلى المواطنين بأفضل صورة ممكنة.
