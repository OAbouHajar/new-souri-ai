# Source: Ministry of Foreign Affairs and Expatriates
# Category: Foreign Affairs
# Date: 2025-12-07
# Link: https://t.me/syrianmofaex1/2038

اجتمع وزير الخارجية والمغتربين السيد أسعد حسن الشيباني بنظيره النرويجي السيد إسبن بارث إيدي في العاصمة القطرية الدوحة، وذلك على هامش أعمال منتدى الدوحة 2025.
وبحث الجانبان سبل تعزيز العلاقات الثنائية وتطويرها، كما استعرض الوزيران مستجدات الأوضاع الإقليمية والدولية ذات الاهتمام المشترك.
