بهدف متابعة تطوير الهيكل التنظيمي، استقبل وزير الاتصالات وتقانة المعلومات السيد "عبد السلام هيكل" وزير التنمية الإدارية السيد "محمد حسان السكاف" لمناقشة الإطار العام للهيكل التنظيمي.

واستعرض الجانبان آليات إقرار الهيكل التنظيمي لتعزيز كفاءة العمل المؤسسي، وتحقيق التكامل بين الأدوار والمسؤوليات عبر الوحدات الإدارية والفنية بالوزارة.

كما أكدا على ضرورة استثمار الكوادر البشرية عبر تنمية المهارات وزيادة الخبرات، باعتبارهم حجر الأساس لأي تحول مؤسسي ناجح ومستدام.
