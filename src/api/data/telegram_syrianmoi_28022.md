# Category: Interior
# Date: 2026-01-03

استعرضت وزارة الداخلية أسطول الآليات الجديدة لوحداتها في مدينة حلب، ويشمل الاستعراض آليات الشرطة والمرور والأمن الداخلي وأمن الطرق ووحدات المباحث الجنائية والحماية والأمن الدبلوماسي، وذلك ضمن جهود الوزارة لتعزيز قدراتها الميدانية والارتقاء بمستوى الجاهزية والكفاءة.
