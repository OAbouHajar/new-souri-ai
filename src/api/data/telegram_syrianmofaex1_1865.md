# Category: Foreign Affairs
# Date: 2025-11-13

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة البريطانية لندن وزير الدولة لشؤون الشرق الأوسط البريطاني السيد هامش فولكنر، وذلك في إطار الزيارة الرسمية التي يقوم بها السيد الوزير إلى المملكة المتحدة.
وجرى خلال اللقاء بحث سبل تعزيز العلاقات الثنائية بين البلدين ومناقشة التطورات الإقليمية والدولية ذات الاهتمام المشترك.
