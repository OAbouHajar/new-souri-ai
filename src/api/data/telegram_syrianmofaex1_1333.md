# Category: Foreign Affairs
# Date: 2025-09-03
# Link: https://t.me/syrianmofaex1/1333

التقى وزير الخارجية والمغتربين السيد أسعد حسن الشيباني وكيلة الأمين العام للأمم المتحدة والأمينة التنفيذية للإسكوا السيدة رولا دشتي، وذلك لبحث أوجه التعاون المشترك بين سوريا ولجنة الأمم المتحدة الاقتصادية والاجتماعية لغربي آسيا (الإسكوا).

وخلال اللقاء استعرض الجانبان الجهود الأممية لدعم خطط التنمية وإعادة الإعمار في سوريا، كما أكد الوزير الشيباني حرص الحكومة على تعزيز الشراكة مع المنظمات الأممية.
