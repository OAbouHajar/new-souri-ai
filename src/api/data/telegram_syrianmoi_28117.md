متزعم الشبكة "ع.ع" وأفرادها "ه.غ"، و"أ.ت"، و"ع.س"، الذين أُلقي القبض عليهم خلال العملية الأمنية التي نفّذتها وحدات إدارة مكافحة المخدرات بالتعاون مع مديرية الأمن الداخلي في مدينة حماة، والتي استهدفت شبكة متخصصة في تصنيع وتهريب المواد المخدّرة.
