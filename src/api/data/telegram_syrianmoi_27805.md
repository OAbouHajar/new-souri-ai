تطلق وزارة الداخلية السورية تطبيقها الرقمي الجديد "صوتك وصل" لتسهيل وصول المواطنين إلى خدماتها الرسمية بسرعة وسهولة، ومن أي مكان.

يتيح التطبيق:
- الاستعلام عن منع السفر ومعرفة وضعك القانوني قبل أي سفر.
- متابعة حالة الموقوفين بشفافية وسرعة.
- تقديم الشكاوى ومتابعتها بشكل مباشر.
