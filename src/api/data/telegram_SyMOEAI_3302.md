في إطار جهود وزارة الاقتصاد والصناعة الرامية إلى تطوير البيئة التشريعية وتعزيز جاذبية مناخ الأعمال، تواصل اللجنة المكلّفة بدراسة وإعادة تعديل أحكام قانون الشركات الصادر بالمرسوم التشريعي رقم /29/ لعام 2011 عقد اجتماعاتها الدورية بإشراف الإدارة العامة للتجارة الداخلية وحماية المستهلك، ومنذ انطلاق أعمال اللجنة بتاريخ 3 أيلول 2025، بلغ عدد الجلسات المنعقدة حتى الآن 13 جلسة.
