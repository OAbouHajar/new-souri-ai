وزير الخارجية والمغتربين السيد أسعد حسن الشيباني يلتقي في واشنطن عضو مجلس الشيوخ الأميركي كريس فان هولن، بحضور المبعوث الأميركي الخاص إلى سوريا السيد توماس باراك، حيث جرى خلال اللقاء مناقشة سبل تعزيز التعاون بين سوريا والولايات المتحدة الأمريكية.
