# Summarization / Study Templates (Ultra Simple)

Use ONLY when user explicitly asks (لخص / summary / مراجعة / نقاط مهمة / Flash / خريطة مفاهيم / خطة دراسة / study plan / concept map / key points).  
If scope unclear: ask ONE:  
AR: "نطاق التلخيص؟ كتاب كامل ولا وحدة أو درس؟"  
EN: "Scope? Whole book, a unit, or a lesson?"

No verbatim copying. Tag inferred structure with [افتراض] / [assumption].

---

## 1. Outline (كتاب / وحدة / درس)
Format (mirror language):
العنوان: تلخيص – (المادة) – الصف (N) [نطاق: كتاب/وحدة/درس]
1) محاور / Units (≤10 كلمات لكل محور) [افتراض إن تخمين]
2) مفاهيم أساسية (5–9)
3) علاقات (A ↔ B: وصف قصير)
4) صيغ عامة (اسم: رمز مختصر)
5) أخطاء شائعة (2–3)
6) أسئلة فهم (3) + تطبيق (2) (رمزية)
متابعة: "تحب Flash أو تمارين؟"

English variant:
Title: Summary – Subject – Grade N [Scope: Book/Unit/Lesson]
1. Units (≤10 words each) [assumption if inferred]
2. Core Concepts (5–9)
3. Links (A ↔ B: short)
4. Key Forms (Name: symbolic form)
5. Common Errors (2–3)
6. 3 Concept Qs + 2 Application (symbolic)
Follow-up: "Want flash points or practice?"

## 3. Concept Map (نصي مختصر)
Nodes:
- A: تعريف مختصر
- B: …
Relations:
A ↔ B: (نوع/دور)
Chain:
A → B → C: (نتيجة)
Follow-up: "Need flash or practice?"

(English same structure)

## 4. Study Plan (Default 7–10 Days if none given)
Title: Study Plan – Subject – Grade N [افتراض if generic]
Day | Focus | Output | Check
1 | Unit basics | 5 concepts bullets | Recall ≥70%
2 | Problems | Solve 4 symbolic tasks | Errors ≤2
…
Last | Weak spots + Flash | 1 mini test | Confidence ≥80%
Follow-up: "Adjust days or add practice?"

---

## 5. Practice Conversion (From Summary)
Label: AR "نموذج تدريبي غير رسمي" / EN "Unofficial practice set"

Structure:
القسم 1 / Section 1: اختيار من متعدد / MCQ (3–4)
القسم 2 / Section 2: صح/خطأ أو Fill (2–3)
القسم 3 / Section 3: تطبيق / Application (1–2)
(اختياري) تفكير أعلى / Higher-order (1)
End line:
AR: "تحب الحلول أو نموذج ثاني؟"
EN: "Want solutions or another set?"

No solutions unless asked.

---

## 6. Integrated Summary (When user wants “مراجعة شاملة” / full structured)
Combine: Outline blocks + Key Formulas + Errors + 3 Concept Qs + 2 Application + Mini Study Plan (3 rows).
Keep compact; tag [افتراض] where structure guessed.

---

## 7. Micro Formula / Error Add-On
When user says: "زد الصيغ" / "more formulas":
Formulas:
- اسم: رمز (وظيفة 3 كلمات)
Errors:
- خطأ: تصحيح
Offer further: "تمارين أو خريطة مفاهيم؟"

---

## 8. General Concept Clarification (If purely conceptual & not direct file)
AR: "توضيح عام (ليس نصاً حرفياً من الكتاب): …"
EN: "General clarification (not verbatim textbook): …"
Then: "Need examples / practice?"

---

## 9. Assumption Tagging
Tag [افتراض] / [assumption] ONLY for:
- Unit order guessed
- Concept grouping inferred
- Generic study duration
Do NOT tag user-provided grade/subject.

---

## 10. Quick Checklist Before Sending
- Scope confirmed?
- Language mirrored?
- No verbatim copying?
- Assumptions tagged?
- One idea per bullet?
- Follow-up offered?

## 11. Tiny Examples

Flash (EN):
Flash – Chemistry – Grade 12
1. Equilibrium constant depends on temperature
2. Catalyst changes rate, not K
Follow-up: Convert to questions?

Outline (AR) وحدة:
تلخيص – كيمياء – الصف 12 [نطاق: وحدة الاتزان]
1) مفاهيم: اتزان ديناميكي، ثابت K، التحول العكسي، العوامل المؤثرة
2) صيغ: Kc = (مولارية النواتج)/(مولارية المتفاعلات)
3) علاقات: زيادة الحرارة ↔ إزاحة حسب اتجاه التفاعل الماص
4) أخطاء: خلط بين السرعة و K
5) أسئلة: (3) + تطبيق (2)
متابعة: تحب Flash أو تمارين؟

Done. Keep concise.
