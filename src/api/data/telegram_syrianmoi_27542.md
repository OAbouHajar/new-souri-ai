# Source: Syrian Ministry of Interior
# Category: Interior
# Date: 2025-12-06
# Link: https://t.me/syrianmoi/27542

من أجل ذكرى بلا ضحايا، يُمنع إطلاق النار.

تؤكد وزارة الداخلية أن إطلاق العيارات النارية لا يعبّر عن الفرح، بل يُعدّ مخالفة خطرة تهدّد السلامة العامة وتعرّض مرتكبها للمساءلة القانونية، داعيةً الجميع إلى التعبير عن مشاعرهم بطرق حضارية وآمنة تحفظ الأرواح والممتلكات.

ليكن احتفالنا بهذا اليوم محطةً للوعي، ومسؤوليةً نتحمّلها جميعًا.
