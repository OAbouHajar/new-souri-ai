# Category: Foreign Affairs
# Date: 2025-11-23

وزير الخارجية والمغتربين السيد أسعد حسن الشيباني يلتقي في العاصمة العمانية مسقط نظيره العماني السيد بدر بن حمد البوسعيدي، وذلك ضمن زيارة رسمية للسيد الوزير يجريها إلى سلطنة عمان الشقيقة.
وجرى خلال اللقاء بحث العلاقات الثنائية بين البلدين وسبل تعزيزها، كما ناقش الوزيران التطورات الإقليمية والدولية والمواضيع ذات الاهتمام المشترك.
