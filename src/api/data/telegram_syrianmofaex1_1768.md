# Date: 2025-11-04

استقبل وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة دمشق رئيس معهد الشرق الأوسط السيد ستيوارت جونز.
وخلال اللقاء تناول الجانبان تطورات الأوضاع في منطقة الشرق الأوسط، والعلاقات السورية الأمريكية وطرق تعزيزها، بالإضافة للتعاون المشترك في المجالات المختلفة أبرزها إلغاء قانون قيصر.
