بمناسبة ذكرى التحرير الأولى أقامت وزارة الخارجية والمغتربين حفل استقبال رسمي للسفراء ورؤساء البعثات الدبلوماسية المعتمدين لدى الجمهورية العربية السورية، وذلك بحضور وزير الخارجية والمغتربين السيد أسعد حسن الشيباني.
