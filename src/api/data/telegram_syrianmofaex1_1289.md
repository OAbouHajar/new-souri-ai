# Category: Foreign Affairs
# Date: 2025-08-25

التقى فخامة الرئيس أحمد الشرع في العاصمة دمشق، وفدًا أمريكيًا رفيع المستوى يرأسه مبعوث الولايات المتحدة الأمريكية الخاص إلى سوريا السيد توماس باراك، وضم الوفد عضو مجلس الشيوخ السيناتور جين شاهين، وعضو مجلس النواب جو ويلسون، كما حضر اللقاء مدير إدارة الشؤون الأمريكية في وزارة الخارجية والمغتربين السيد قتيبة إدلبي.
