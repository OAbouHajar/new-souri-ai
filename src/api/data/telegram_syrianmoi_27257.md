# Source: Syrian Ministry of Interior
# Category: Interior
# Date: 2025-11-25
# Link: https://t.me/syrianmoi/27257

العملية الأمنية التي نفّذتها قيادة الأمن الداخلي في محافظة حلب، بالتعاون مع جهاز الاستخبارات العامة، التي استهدفت خلايا تنظيم داعش الإرهابي في منطقة عفرين شمال غربي المحافظة، أسفرت عن اعتقال أفراد الخلية بعد ثبوت تورطهم في التخطيط لعمليات تهدد أمن المواطنين واستقرار المنطقة.
