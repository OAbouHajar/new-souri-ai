اجتمع وزير الاتصالات وتقانة المعلومات المهندس "حسين المصري" مع ممثلي شركة "نجمة الأفق لتقانة المعلومات" المتخصصة في مجال تقانة المعلومات والتوقيع الرقمي، تمّ خلال اللقاء مناقشة سبل التعاون الممكنة بين الوزارة والشركة في مجال تطوير تقانة المعلومات، وتعزيز استخدام التوقيع الرقمي في مختلف القطاعات، كما تم استعراض الإمكانيات المتاحة لدعم التحول الرقمي وتطوير البنية التحتية التقنية لضمان توفير خدمات إلكترونية آمنة وموثوقة.
