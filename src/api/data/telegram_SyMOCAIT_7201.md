عقد وزير الاتصالات وتقانة المعلومات المهندس "حسين المصري" اجتماعين مع مديري المديريات في وزارة الاتصالات وتقانة المعلومات، لاستعراض الأعمال التي نفذتها المديريات منذ تحرير سوريا من النظام البائد وحتى تاريخه، وعرض خطة العمل للمرحلة القادمة، وناقش معهم التصور المبدأي للهيكلية التنظيمية الجديدة التي ستدير قطاع الاتصالات في سورية.
