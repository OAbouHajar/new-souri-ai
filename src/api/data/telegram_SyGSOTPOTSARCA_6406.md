معاون الأمين العام لرئاسة الجمهورية 
يجتمع باللجنة المركزية لتنظيم الآليات الحكومية

عقد معاون الأمين العام لرئاسة الجمهورية الأستاذ علي كدة اجتماعا مع أعضاء اللجنة المركزية لتنظيم الآليات الحكومية للاطلاع على مستجدات عمل اللجنة واستعراض أبرز المهام المحققة.

نوقشت خلال الاجتماع التحديات التي واجهت اللجنة خلال مراحل العمل السابقة مع طرح المقترحات والحلول المناسبة لتحسين كفاءة العمل وتسريع وتيرته.

وأشارت اللجنة إلى بدء خطوات عملية لربط الجهات الحكومية المعنية عبر برنامج إلكتروني موحد لإدارة وتسجيل السيارات الحكومية وحفظ بياناتها.
