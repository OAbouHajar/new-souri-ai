# Category: Foreign Affairs
# Date: 2025-10-15
# Link: https://t.me/syrianmofaex1/1655

فخامة رئيس الجمهورية السيد أحمد الشرع، يرافقه وزير الخارجية والمغتربين السيد أسعد حسن الشيباني، ووزير الدفاع اللواء مرهف أبو قصرة، ورئيس الاستخبارات العامة السيد حسين السلامة، يلتقي الرئيس الروسي السيد فلاديمير بوتين وكبار المسؤولين في قصر الكرملين بالعاصمة موسكو.
وتأتي الزيارة في إطار تعزيز العلاقات بين دمشق وموسكو، وتشمل مباحثات موسعة حول عدة مجالات.
