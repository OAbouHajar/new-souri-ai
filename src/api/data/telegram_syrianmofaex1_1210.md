# Source: Ministry of Foreign Affairs and Expatriates
# Category: Foreign Affairs
# Date: 2025-08-06

استقبل فخامة رئيس الجمهورية العربية السورية السيد أحمد الشرع بحضور وزير الخارجية والمغتربين السيد أسعد حسن الشيباني في العاصمة دمشق المبعوث الخاص للولايات المتحدة الأمريكية إلى سوريا السيد توماس باراك، حيث تناول اللقاء آخر المستجدات السياسية والأمنية وسبل دفع العملية السياسية قدمًا بما يضمن احترام سيادة سوريا ووحدة أراضيها.
