بقلوب مؤمنة بقضاء الله وقدره تنعى وزارة الاتصالات وتقانة المعلومات 
المهندس "لؤي أحمد دحكول" معاون مدير عام المؤسسة السورية للبريد، إثر حادث سير أليم أثناء تأديته عمله.
نسأل الله أن يتغمده بواسع رحمته ويغفر له ويلهم أهله وذويه وزملاءه الصبر والسلوان.
