اختتام المؤتمر الوطني للإعاقة «التمكين والشمول والدمج»، برعاية الأمانة العامة لرئاسة الجمهورية وتنظيم وزارة التعليم العالي، بحضور معاون الأمين العام السيد "علي كدة"، ووزير التعليم العالي الدكتور "مروان الحلبي"، وبمشاركةٍ وزاريةٍ واسعةٍ أكدت تحويل الدراسات والتوصيات إلى سياساتٍ عمليةٍ داعمةٍ لذوي الإعاقة.
